# Kalign

Kalign is a fast multiple alignment program for protein and nucleotide sequences.


# Install

``` bash
git clone https://github.com/TimoLassmann/kalign.git 
cd kalign
./autogen.sh
make 
make check 
make install 
```

# Usage

``` sh

kalign -i <input sequences> -o <output alignment file> 
```
